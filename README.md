# FIFA World Cup Goal Analysis

## Overview
This project analyzes FIFA World Cup match event data to compare goals scored by the first 20 teams in the dataset.

## Skills Used
- Google Sheets
- Data Filtering
- Pivot Tables
- Data Visualization
- GitHub

## Process
- Imported a FIFA World Cup match events dataset.
- Filtered the data to include only goal events.
- Created a pivot table to count goals by team.
- Built a chart to visualize team scoring totals.

## What I Learned
This project helped me practice filtering data, summarizing information with pivot tables, and creating visualizations from real-world sports data.
